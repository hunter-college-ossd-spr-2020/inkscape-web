Inkscape efficace: Réussir ses dessins vectoriels

  • Auteur : Cédric Gémy
  • Format : broché, 280 pages
  • Éditeur/Date : Eyrolles / février 2009
  • ISBN-10 : 2-21-212425-2
  • ISBN-13 : 978-2-2121-2425-5
  • Site internet : http://www.eyrolles.com

Il s'agit du deuxième livre de Cédric après le succès commercial de « GIMP Efficace ». Les 120 premières page du livre, décrivant les outils de base, la couleur et les transformations, sont adaptées à un public débutant. Les autres parties vient plutôt les utilisateurs professionnels et décrivent en profondeur comment améliorer leur efficacité avec Inkscape, travailler pour l'impression ou faire des sites orientés SVG (même si du code est nécessaire). Lorsque c'est possible, Cédric essaie de donner les conseils pour les utilisateurs d'Adobe Illustrator.

Inkscape: Apprenez, pratiquez, créez

  • Auteur : Elisa de Castro Guerra
  • Format : broché, 256 pages
  • Éditeur/Date : Pearson France / octobre 2008
  • ISBN-10 : 2-74-402276-4
  • ISBN-13 : 978-2-7440-2276-0
  • Site internet : http://www.pearson.fr

Cet ouvrage contient des explications complètes sur l'utilisation d'Inkscape, agrémentées de nombreux exercices qui vous donneront les clef le maîtriser. Cette nouvelle édition, basée sur la version 0.46 d'Inkscape, présente également quelques unes des nouveautés de la version 0.47.

Inkscape : Premiers pas en dessin vectoriel

  • Auteur : Nicolas Dufour, avec la contribution d'Elisa de Castro Guerra
  • Format : broché, 376 pages
  • Éditeur/Date : Eyrolles / mai 2009
  • ISBN-10 : 2-21-212444-9
  • ISBN-13 : 978-2-212-12444-6
  • Site internet : http://www.eyrolles.com

En 376 pages, apprenez à maîtriser efficacement ce logiciel de dessin vectoriel libre à la portée de tous. Des outils simples et avancés, des fonctionnalités de combinaisons, d'effets, des aides vous permettront de mieux organiser vos documents. Le plus : 9 études de cas pour vous entraîner pas à pas.

  • Auteur : Bethany Hiitola
  • Format : broché, 316 pages
  • Éditeur/Date : Packt Publishing / novembre 2010
  • ISBN-10 : 1-84-951268-X
  • ISBN-13 : 978-1-84951-268-8
  • Site internet : http://www.packtpub.com

Learn to build your first website design using Inkscape. Create web page and desktop wallpapers with repeating pattern backgrounds and swirling designs. Incorporate icons and interactive maps on your website. Style and graphically manipulate text — from simple headings to shadowing, following paths, reflections, 3D effects, and more. Enhance your web pages using flowcharts, diagrams, and site maps and learn how to export them. Spruce it all up using animations.

Inkscape: Guide to a Vector Drawing Program

  • Auteur : Tavmjong Bah
  • Format : broché (480 pages), PDF
  • Langues: anglais, espagnol, italien, français
  • Éditeur/Date : SourceForge Community Press / novembre 2009
  • ISBN-10 : 0-13-705173-5
  • ISBN-13 : 978-0-13-705173-1

Inkscape: Guide to a Vector Drawing Program, Third Edition, is the guide to the Inkscape program, with coverage of versions 0.46 and 0.47. The Book of Inkscape

  • Auteur : Dmitry Kirsanov
  • Format : broché, 476 pages
  • Éditeur/Date : No Starch Press / septembre 2009
  • ISBN-10 : 1-59327-181-6
  • ISBN-13 : 978-1-59327-181-7
  • Site internet : http://nostarch.com

In The Book of Inkscape, core Inkscape developer Dmitry Kirsanov shares his design experience and knowledge of Inkscape's inner workings as he walks you through the basics of using the program: drawing, working with objects, transformations and styling, adding text and shapes, and more. Kirsanov couples his detailed explanations with step-by-step tutorials that show you how to create business cards, animations, and technical and artistic drawings.

Learn SVG: The Web Graphics Standard

  • Auteurs : Jon Frost, Stefan Goessner and Michel Hirtzler, revisited by Robert DiBlasi and Tobias Reif
  • Format : broché (316 pages) et eBook
  • Langues: anglais, français
  • Éditeur/Date : SVG / 2003
  • ISBN-10 : 2-95-208780-6 (english?)
  • Site internet : http://www.learnsvg.com/

Learn SVG: The Web Graphics Standard is a beginner-level book that assumes basic knowledge of Hypertext Markup Language (HTML). Learn SVG is presented in a workbook-like format that lends itself to people who want both a solid foundation in the main aspects of SVG and who are willing to get their hands dirty with the fertile soil that SVG has to offer. The full challenge of the book is not for the faint of heart but the authors have made sure to thoroughly cover all the bases using concrete examples.

The reader will move on to more advanced topics of adding style, gradients and filter effects, menu creation, animation, script-based interactivity, publishing, extensibility, and a cutting-edge SVG Web application.
