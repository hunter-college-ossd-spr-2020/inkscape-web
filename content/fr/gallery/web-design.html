Ces exemples explorent des sites internet conçus avec l'aide d'Inkcape.

Moonshine, par Jakub « Jimmac » Steiner

{# Link to http://www.moonshine.cc/ removed; parked #}

Inkscape allows to export multiple objects into bitmaps, naming the exported PNGs according to the respective object ids. The export dialog also allows to select to only export the selected object and hide everything else. Using just those two features along with the possibility to create 'blank' objects with no stroke or fill and grouping objects together gives a web designer a great tool to not only mock up a site layout, but also use the very same drawing for exporting the actual image elements for the final site. The outline view helps to select the 'blank' objects for manipulation.
