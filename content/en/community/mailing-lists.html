Think of mailing lists as a forum but each poster sends their comments by email. Each list covers a topic, so choose yours carefully. Posting your comment in the wrong place won't get the attention you expect.
Also, an NNTP Internet News gateway is available through Gmane. To access it, point your favorite news browser to nntp://news.gmane.org.

inkscape-user

If you are an user and don't know how to do something or have doubts about some feature, this is your place.

inkscape-devel

Use this for discussion of development related issues: compile problems, crashes, misfeatures, etc.
[subscribe] [archives]

inkscape-tester

Use this for discussion of usability, performance, robustness, etc. Also the place to post test results.
[subscribe] [archives]

inkscape-docs

Use this for discussion of official tutorials and the user manual.
[subscribe] [archives]

inkscape-translator

Use this for discussion of translation of Inkscape interface and related topics.
[subscribe] [archives]

inkscape-wiki

Notifications of major changes to wiki pages are cc'd to this list.
[subscribe] [archives]

inkscape-cvs

Note: This list is only used for commits related to the former website; for Inkscape code go to Launchpad.
SVN commit messages for all Inkscape SVN modules are cc'd here (high volume).
[subscribe] [archives]
