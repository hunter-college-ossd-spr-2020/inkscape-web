For casual questions or in-depth discussion, the official chat room is the Jabber (XMPP) conference room inkscape@conference.gristle.org, which is bridged with the channel #inkscape on IRC for bidirectional communication (however IRC users need to register their nicknames at Freenode server otherwise they won't be allowed to talk).
While this channel is primarily geared for developer discussion, we do not discourage users from participating too, however please be conscientious of any development discussions going on. If you do not get an immediate answer to your question, folks may just be focusing on other things; try asking again later, or use the mailing lists or forums instead.

Setting up a Jabber or IRC client

Jabber

First, you need a Jabber client such as Pidgin, iChat, Google Talk or Kopete. For an extensive list, see Wikipedia list of Jabber clients.

Second, set up an account with a server such as gristle.org or myjabber.net, then join the conference room inkscape@conference.gristle.org.

Also see the tutorial Setting up Pidgin (formerly Gaim) for inkscape.gristle.org.

IRC

First you need a IRC client such as BitchX, XChat, Pidgin, Kopete, Empathy or Miranda. For an extensive list, see Wikipedia list of IRC clients. Or use the webchat which doesn't require any installation.

Second, register your nickname at Freenode server, otherwise you won't be allowed to talk.

And third, start the client and point to Inkscape channel writing ‘/join #inkscape’ at the prompt. Some clients accept direct links to channels like this: #inkscape

It's recommended to configure your client to use the utf-8 character set if possible. In many clients, this can be done writing ‘/charset utf-8’ or ‘/set term_type utf-8’ at the prompt.

Jabber? IRC? Why not both?

There is a wide selection of multi-protocol clients which handle both Jabber and IRC protocols, and many others. Using one is useful if you want to manage all your communications from only one software. However don't connect to both networks at the same time and, if you can choose, use Jabber.
