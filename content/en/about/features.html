Inkscape is a professional quality vector graphics software with sophisticated drawing and manipulation tools. It has a comprehensive feature set that is used by professionals and hobbyists around the world.

Notable Features

Drawing tools

At the heart of Inkscape are sophisticated drawing tools that allow freehand drawing or bezier curves.

Node points

Inkscape's node tool enables precision editing of multiple paths at once.

Text tool

Inkscape has on-canvas text editing with precise control over typography.

XML editor

Directly manipulate underlying SVG code to ensure complete W3C compliancy.

Creation

Shape tools

Inkscape has a number of tools for drawing geometric shapes.

Gradients

Inkscape allows easy onscreen editing of complex gradients.

Clones

Inkscape clones are "live" linked copies of objects that change with the original, but can also be transformed independently.

Bitmaps

Import raster graphics by either linking or embedding them, and perform basic editing. You can also create bitmaps of selected objects.

Connectors

Creating organization charts or flow diagrams is easy with Inkscape's connector tool.

Manipulation

Transformations

Inkscape supports a number of object transformations, such as scale, rotate and skew.

Styling

Inkscape has intuitive fill and stroke dialogs that control fill and stroke colour, and stroke styles. The colour palette bar allows quick changes.

Layers

Inkscape layers let you organise elements into a logical structure. You can lock or hide individual layers.

Positioning

Positioning objects in Inkscape is simple with the selector tool, the align & distribute dialog, and z-order operations.

Object grouping

Inkscape lets you group related objects for easier manipulation.
