Users

Inkscape caters to a wide audience

If you are an illustrator, designer, web designer, typographer or just someone who needs to create some vector imagery, Inkscape is for you!

Check Inkscape's main features

  • Several primitives (rectangles, 3D cubes, circles/ellipses, stars/polygons and spirals) allows fast creation of graphics
  • Versatile calligraphic simulation
  • Powerful bezier and spiro curve editing
  • Live path effects (LPE) helps you manipulate your shapes creating complex and unique forms

If you are interested or have any doubts suscribe to the users mailing list.
